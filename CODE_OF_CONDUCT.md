# Code of Conduct

This project follows the
[Contributor Covenant Code of Conduct, v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

Be respectful. Assume good intent. No harassment. No personal attacks. Keep
discussion focused on the code and the problem.

## Reporting

To report a violation, open a private issue or contact the maintainer at the
email listed in `pyproject.toml`. Reports will be handled confidentially.
